# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✓         |

## Reporting a Vulnerability

If you discover a security vulnerability, please open a [GitHub issue](https://github.com/mvareva/focus-timer/issues) or contact the maintainer directly. We'll respond as soon as possible.

Please do not disclose security issues publicly until they have been addressed.
