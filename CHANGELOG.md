# Changelog

All notable changes to this project will be documented here.

## [0.1.0] — 2026-03-08

### Added
- Focus (25 min), Short Break (5 min), and Long Break (15 min) timer presets
- Animated SVG ring progress indicator
- Ambient radial glow that transitions with each preset
- Completion overlay with auto-advance to next preset
- Space bar shortcut to start / pause the timer
- `prefers-reduced-motion` support — all animations disabled when set
- ARIA live regions and labelled controls for accessibility
- Vite + React 18 project scaffold
- GitHub Actions workflow for automatic deployment to GitHub Pages
